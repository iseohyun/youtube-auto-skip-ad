# YouTube Auto Skip Ad (유튜브 광고 자동 건너뛰기 크롬 확장 프로그램)

유튜브 동영상 시청 중 나타나는 광고를 감지하여 백그라운드 탭이나 최소화 상태에서도 자동으로 건너뛰어 주는 가볍고 세련된 크롬 확장 프로그램입니다.

![YT Skipper Preview](icon.png)

## 주요 기능

1. **지능형 광고 탐색 (`content.js`)**: 
   - 웹페이지 리소스를 절약하기 위해 반복적인 루프 대신 **`MutationObserver`**를 활용하여 유튜브 플레이어 내부의 DOM 변경을 실시간으로 감지하고 버튼이 노출되는 즉시 클릭합니다.
   - 플레이어 내부(`.html5-video-player`) 영역으로 탐색을 국한하여 본문 건너뛰기 링크 등 웹 접근성용 페이지 내 다른 요소를 오인하지 않습니다.

2. **강력한 이중 우회 스킵**:
   - **이벤트 에뮬레이션**: 크롬의 자동화 방어 기작(`isTrusted = false` 차단)을 우회하기 위해 `mousedown -> mouseup -> click` 마우스 이벤트를 순차적으로 에뮬레이션합니다.
   - **16배속 쾌속 백업**: 클릭이 금지된 강제 광고나 일시적인 조작 불가 광고의 경우, 광고가 재생되는 즉시 영상 배속을 **16배속**으로 바꾸고 음소거 처리하여 0.5초 이내에 강제 종료시킵니다.

3. **세련된 다크 모드 대시보드 (`popup`)**:
   - 글래스모피즘(Glassmorphism) 스타일이 적용된 트렌디한 다크 테마 대시보드를 제공합니다.
   - **스마트 카운터**: 광고 1개당 정확히 1회씩만 카운팅되도록 상태 변화(Ad Status Flow) 알고리즘을 탑재하여 스킵 횟수가 무한대로 복사되는 현상을 완벽히 방지합니다.
   - 작동 온/오프 토글 스위치 및 통계 초기화 기능을 제공합니다.

## 설치 및 사용 방법

### 1단계: 프로젝트 다운로드
- 이 저장소를 클론하거나 ZIP 파일로 다운로드하여 로컬 드라이브의 적절한 폴더에 압축을 풉니다.

### 2단계: 크롬에 확장 프로그램 등록
1. 구글 크롬 브라우저를 열고 주소창에 `chrome://extensions/`를 입력하여 이동합니다.
2. 우측 상단의 **'개발자 모드 (Developer mode)'** 스위치를 켭니다.
3. 좌측 상단에 나타나는 **'압축해제된 확장 프로그램을 로드합니다 (Load unpacked)'** 버튼을 클릭합니다.
4. 다운로드하거나 클론한 폴더(본 프로젝트 폴더)를 선택하여 등록합니다.

### 3단계: 주소창 우측 고정
1. 크롬 우측 상단의 퍼즐 조각 모양 아이콘(확장 프로그램 메뉴)을 클릭합니다.
2. **YouTube Auto Skip Ad** 옆의 핀 아이콘을 눌러 툴바에 고정합니다.
3. 아이콘을 눌러 대시보드를 열고 유튜브를 새로고침하면 백그라운드 광고 스킵이 즉시 작동합니다.

## 파일 구성

* `manifest.json`: 확장 프로그램의 구조와 권한 설정 메타데이터
* `content.js`: 광고 감지 및 시뮬레이션 클릭/배속 처리 핵심 로직
* `popup.html`: 설정 대시보드 레이아웃
* `popup.css`: 대시보드 다크 테마 디자인 및 애니메이션 스타일시트
* `popup.js`: 대시보드 상태 컨트롤 및 누적 카운트 연동 로직
* `icon.png`: 확장 프로그램 앱 아이콘 이미지
