# YT ad watch & click (유튜브 광고 정속 완독기)

> [!IMPORTANT]
> **Beta Version Notice (베타 버전 안내)**
> 이 확장 프로그램은 현재 **베타 버전(Beta Version)**입니다. 유튜브의 실시간 광고 차단 차단 및 봇 감지 알고리즘 업데이트에 지속적으로 대응 및 패치를 진행 중입니다.

>  * **신뢰할 수 있는 디버거 클릭 (가상 클릭)**
> 합성 이벤트(`isTrusted = false`) 차단은 스크립트가 크롬에게 거짓 사용자 행동을 명령하는 것을 방어하기 위해 설계되었습니다. 따라서 **합성 이벤트를 사용하기 위해서는 '개발자 모드'를 사용**해야합니다. 이 때, isTrusted = true를 만들면, **"'YT' ad watch & click에서 브라우저에 대한 디버깅을 시작함"**이라는 메시지가 출력됩니다(의도된 출력).

유튜브 동영상 시청 중 나타나는 광고를 창작자 후원을 위해 **정속으로 완전히 시청**하되, 매번 마우스로 스킵 버튼을 직접 찾아 클릭해야 하는 번거로움을 대행해 주는 프로그램입니다.
1. 광고 배속이나 가속을 임의로 사용하지 않습니다.
2. 숨겨진 상태의 버튼을 강제로 찾아 클릭하지 않습니다.
3. 광고 필수 시청 완료 후, 의도된 스킵 타이밍에 맞추어 클릭을 대행합니다.

## 🛡️ 봇 탐지 방어 기믹 및 대응 솔루션 (Anti-Bot Defenses)
1. **유튜브의 봇 감지 방어 기믹**
   * **버튼 영역 미진입 기믹**: 마우스 포인터가 버튼 영역으로 진입하는 호버 단계(`mouseover`, `mouseenter`) 기록이 있어야 클릭을 유효한 것으로 인정합니다.
   * **휴머노이드 마우스 시뮬레이션**: 지연 클릭 옵션이 활성화된 경우 무작위 대기 시간(1.0~2.5초)을 적용하며, 마우스가 먼저 버튼 영역에 진입한 후 호버 상태를 유지한 채 클릭합니다. 또한, 매번 동일한 중앙 좌표가 아닌 버튼 영역 내의 20%~80% 범위의 무작위 분산 좌표를 계산하여 클릭합니다.
2. **화면 최소화(Minimization) 상태 렌더링 중단 우회**
   * **유튜브의 방어 기믹**: 크롬 창이 최소화되면 브라우저는 GPU/렌더링을 일시 중단하므로 요소의 실제 픽셀 크기(`getBoundingClientRect`)가 0이 됩니다. 일반적인 스킵 툴은 크기가 0인 버튼을 '숨김 요소'로 착각하여 대기 상태에 머물게 됩니다.
   * **우회 솔루션**: 픽셀 크기가 0이어도 스위칭 모드로 동작하여 탭 상태를 추적하고, 창 활성화 시 즉시 스킵 로직을 정상 재개합니다.

## 주요 기능 및 대시보드 구성

### 1. 영상 & 광고 시청 계기판 (그룹 1)
* **총 영상시청시간 / 현재 영상시청시간**: 사용자가 시청한 전체 영상의 누적 및 세션별 시청 시간을 표기합니다.
* **총 광고시청시간 / 현재영상 광고시청시간**: 누적 및 세션별로 광고를 시청한 시간을 기록합니다. 광고 재생 중에는 해당 카드가 더 밝은 보라색으로 강조 표시됩니다.

### 2. 실시간 스킵 필터 진단판 (그룹 2)
* **버튼 감지**: DOM 상에서 광고 건너뛰기(스킵) 버튼 요소를 발견하면 켜집니다.
* **버튼 위치/크기**: 스킵 버튼의 실제 픽셀 너비와 높이 영역이 감지되면 켜집니다.
* **버튼 보임**: CSS 속성상 숨겨진 상태가 아니며 화면에 실제 노출되고 있는 상태일 때 켜집니다.
* **광고 재생 중**: 광고 동영상이 재생 상태일 때 켜집니다. 동영상이 일시정지되면 꺼집니다.
* **필수 시청 완료**: 광고 필수 시청 카운트다운(5초 등)을 추적하며, 카운트다운이 종료되면 녹색으로 완료 표시가 켜집니다.
* **버튼 불투명**: 스킵 버튼의 CSS 투명도(opacity)가 활성화 수준(0.1 초과)으로 선명해지면 켜집니다.

### 3. 실행 및 스킵 설정 (그룹 3)
* **클릭 수행**: 버튼을 누르면 즉시 유튜브 플레이어의 스킵 버튼 강제 클릭 대행을 수동으로 수행합니다.
* **지연 클릭**: (기본값: ON) 활성화 시 스킵 조건 충족 후 1.0~2.5초 무작위 대기 후 클릭합니다. 비활성화 시 0초 즉시 클릭을 수행합니다.
* **랜덤 위치 클릭**: (기본값: ON) 활성화 시 스킵 버튼 영역 내부의 무작위 분산 좌표를 클릭합니다. 비활성화 시 버튼 정중앙 좌표를 정확히 클릭합니다.
* **가상 클릭**: (기본값: ON) 활성화 시 Chrome Debugger API를 경유하여 신뢰할 수 있는 하드웨어 클릭(`isTrusted=true`)으로 스킵합니다. 비활성화 시 일반 브라우저 합성 이벤트(`isTrusted=false`)를 사용하여 즉시 클릭합니다.

### 4. 로그 복사 및 초기화 (그룹 4)
* **현재 상태 복사**: 대시보드의 활성 설정 상태, 재생 정보, 최근 스킵 타임라인 로그, 최근 시청 역사(최근 5건), 내부 시스템의 예외/오류 스택을 통합하여 한 번에 클립보드에 복사합니다.
* **로그 초기화**: 과거 스킵 시각 로그 및 수집된 시스템 오류 내역을 삭제합니다.
* **시청시간 초기화**: 누적 시청 시간 통계 수치를 0초로 재설정합니다.
* **대시보드 초기화**: 대시보드 설정을 초기값(지연 클릭 ON, 랜덤 위치 ON, 가상 클릭 OFF)으로 일괄 복원합니다.

## 설치 및 사용 방법

### 1단계: 프로젝트 다운로드
* 이 저장소를 다운로드하여 로컬 드라이브의 적절한 폴더에 압축을 풉니다.

### 2단계: 크롬에 확장 프로그램 등록
1. 구글 크롬 브라우저를 열고 주소창에 `chrome://extensions/`를 입력하여 이동합니다.
2. 우측 상단의 **'개발자 모드 (Developer mode)'** 스위치를 켭니다.
3. 좌측 상단에 나타나는 **'압축해제된 확장 프로그램을 로드합니다 (Load unpacked)'** 버튼을 클릭합니다.
4. 다운로드하거나 클론한 폴더(본 프로젝트 폴더)를 선택하여 등록합니다.

### 3단계: 주소창 우측 고정
1. 크롬 우측 상단의 퍼즐 조각 모양 아이콘(확장 프로그램 메뉴)을 클릭합니다.
2. **YT ad watch & click** 옆의 핀 아이콘을 눌러 툴바에 고정합니다.
3. 아이콘을 눌러 대시보드를 열고 유튜브를 새로고침하면 대행 작동이 즉시 시작됩니다.
