# YouTube Ad Full Watch (Beta) (유튜브 광고 정속 완독기)

> [!IMPORTANT]
> **Beta Version Notice (베타 버전 안내)**
> 이 확장 프로그램은 현재 **베타 버전(Beta Version)**입니다. 유튜브의 실시간 광고 차단 차단/봇 감지 알고리즘 업데이트에 지속적으로 대응 및 패치를 진행 중입니다.

유튜브 동영상 시청 중 나타나는 광고를 창작자 후원을 위해 **정속으로 성실하게 완전히 시청**하되, 매번 마우스나 리모컨을 찾아 스킵 버튼을 직접 클릭해야 하는 귀찮음만 대행해 주는 가볍고 영리한 크롬 확장 프로그램입니다.

![YouTube Ad Full Watch Preview](icon.png)

## 개발 철학 및 역사 (History)

이 확장 프로그램은 유튜브 광고 생태계와 사용자 편의성 간의 공존을 고민한 결과물입니다.

1. **정속 완주 정책 (100% Normal Speed)**
   * 과거 광고 배속을 높이는 방식을 적용해 보았으나, 유튜브 서버에서 광고 스킵 의심 및 차단 배너가 자주 발생하는 이슈가 있었습니다. 따라서 광고는 우회나 꼼수 없이 **원래 속도(정속)로 안전하게 끝까지 감상**합니다.
2. **인간적인 딜레이 선심 정책 (1~2.5초 지연)**
   * 광고 건너뛰기 버튼이 활성화되자마자 0초 만에 바로 광속 클릭을 하면 자동화 매크로로 감지되어 필터링될 가능성이 높습니다. 유튜브 측에도 1~2초 더 광고를 봐주는 선심을 베풀며 자연스러운 타이밍에 지연 클릭을 수행합니다.
3. **사용자 편의 중심의 인터페이스**
   * 광고가 무지하게 많이 나오는 요즘, 침대나 소파에 편하게 누워 있다가 건너뛰기 버튼 하나 누르자고 리모컨이나 마우스를 매번 찾는 번거로움을 해결하기 위한 '클릭 대행 비서'의 역할을 지향합니다. 광고를 정상적으로 봐주는 대신, 스킵 과정에서 발생하는 불필요한 짜증과 피로감을 획기적으로 줄여줍니다.

## 🛡️ 봇 탐지 방어 기믹 및 대응 솔루션 (Anti-Bot Defenses)

이 확장 프로그램은 유튜브가 자동화 봇 및 매크로 스킵을 차단하기 위해 사용하는 여러 방어 기믹들을 정교한 프로그래밍 기법으로 모두 우회합니다.

1. **마우스 이벤트 좌표 미세 무작위화 및 호버-클릭 시차 에뮬레이션**
   * **유튜브의 방어 기믹**: 스크립트로 생성한 가상 클릭(`isTrusted = false`)이나 클릭 좌표가 지정되지 않은 이벤트(clientX/Y가 `0`)를 차단하며, 특히 마우스 포인터가 버튼 영역으로 진입하는 호버 단계(`mouseover`, `mouseenter`) 기록이 없거나 클릭 이벤트와 동일한 타임스탬프(밀리초)에 동시에 날아오는 기계적 패턴을 필터링함.
   * **우회 솔루션**: 총 지연 대기 시간(1.0~2.5초 사이 무작위) 내에서 마우스가 먼저 단추 영역에 진입해 머무는 호버 시간(0.15~0.5초 사이 무작위)을 뺍니다. 이후 **먼저 호버 이벤트를 발송하여 버튼 위에 마우스 포인터를 안착시킨 다음, 지정된 대기 시차가 흐른 뒤 최종 클릭 이벤트를 실행하는 자연스러운 비동기식 마우스 모션**을 구현했습니다. 또한 스킵 버튼 내부 20%~80% 범위의 매번 다른 무작위 난수 좌표를 주입하여 패턴을 완전 무력화합니다.
2. **카운트다운 중 무분별한 클릭 스팸 감지 차단**
   * **유튜브의 방어 기믹**: 스킵 버튼 컨테이너는 화면에 존재하나 아직 건너뛰기가 불가능한 카운트다운 도중 강제로 연속 클릭을 전송할 경우 매크로로 인지하여 즉시 재생을 정지 및 차단함.
   * **우회 솔루션**: 현대식 버튼(`.ytp-ad-skip-button-modern`)의 투명도가 0.8 이하이거나 보이지 않을 때는 클릭을 예약하지 않고 정속 완독 대기합니다. 추가로, 스킵 대상 요소의 텍스트에 숫자(0-9)가 1자라도 포함되어 있다면 **100% 카운트다운 상태로 판정하여 클릭 시도를 원천 패스(continue)**하고 카운트다운이 완전히 종료될 때까지 대기합니다.
3. **화면 최소화(Minimization) 상태 렌더링 중단 우회**
   * **유튜브의 방어 기믹**: 크롬 창이 최소화되면 브라우저는 GPU/렌더링을 멈추기 때문에 요소의 실제 픽셀 크기(`offsetWidth/offsetHeight`)가 `0`으로 바뀜. 일반 스킵 툴은 크기가 0인 버튼을 '숨김 요소'로 착각하여 스킵을 진행하지 못함.
   * **우회 솔루션**: 렌더링이 일시 정지되더라도 CSS 계산된 스타일(`getComputedStyle(btn)`)을 분석하여 `display !== 'none'` 및 `visibility !== 'hidden'` 조건식을 백업 검증하여 최소화 및 백그라운드 탭에서도 클릭 대행을 정상 지속합니다.
4. **구버전 스크립트의 잔여 API 예외 스택 제거**
   * **유튜브의 방어 기믹/부작용**: 개발자 모드에서 확장 프로그램을 업데이트하거나 새로고침하면, 이전에 주입된 구버전 스크립트가 유실된 API 컨텍스트를 호출하면서 무한 런타임 예외 스택 에러가 개발자 콘솔에 누적되어 비정상 동작으로 차단될 위험이 있음.
   * **우회 솔루션**: 전체 스크립트를 즉시 실행 함수(IIFE)로 감싸 완전히 격리하고, `chrome.runtime.id`를 확인하는 `isContextValid()` 헬퍼를 도입하여 연결이 유실된 구버전 스크립트는 실시간 옵저버와 setInterval 타이머를 스스로 해제하고 메모리에서 소멸하도록 자가 진단 폐기 시스템을 구축했습니다.

## 주요 기능

1. **정교한 지연 클릭 자동화 (`content.js`)**
   * **`MutationObserver`**를 활용하여 웹페이지 리소스를 거의 점유하지 않고 플레이어 내부 DOM 변경을 실시간 감시합니다.
   * 건너뛰기 버튼 활성화 후, `1.0초 ~ 2.5초` 사이의 랜덤한 인간다운 지연 시간(delay) 후에 클릭을 대행합니다.

2. **실제 마우스 클릭 및 호버 시차 에뮬레이션 (봇 차단 완벽 우회)**
   * 유튜브의 자동화 도구 감지 필터(Trusted Event 및 마우스 궤적 검증)를 우회하기 위해, 스킵 버튼의 화면 좌표(`clientX/Y`, `screenX/Y`)를 실시간으로 계산하여 마우스 포커싱, 호버링(`mouseover` ➡️ `mouseenter`), 그리고 `150ms ~ 500ms`의 자연스러운 시차를 둔 누름/뗌(`mousedown` ➡️ `mouseup` ➡️ `click`) 이벤트를 순차적으로 에뮬레이션합니다. 

3. **실시간 자동 클릭 로그 기록 기능**
   * 광고가 성공적으로 건너뛰어질 때마다 클릭이 처리된 정확한 시스템 시간(타임스탬프)을 내부 저장소에 기록합니다.

4. **세련된 다크 모드 대시보드 (`popup`)**
   * 글래스모피즘(Glassmorphism) 다크 테마 대시보드를 제공하여 작동 상태 토글, 누적 자동 클릭 횟수, 실제 비디오 시청 시간, 광고 대기 시간을 실시간 통계로 보여줍니다.
   * **최근 자동 클릭 기록** 패널을 하단에 탑재하여 최근 스킵된 5개의 타임스탬프를 실시간으로 모니터링할 수 있습니다.

## 설치 및 사용 방법

### 1단계: 프로젝트 다운로드
* 이 저장소를 다운로드하여 로컬 드라이브의 적절한 폴더에 압축을 풉니다.

### 2단계: 크롬에 확장 프로그램 등록
1. 구글 크롬 브라우저를 열고 주소창에 `chrome://extensions/`를 입력하여 이동합니다.
2. 우측 상단의 **'개발자 모드 (Developer mode)'** 스위치를 켭니다.
3. 좌측 상단에 나타나는 **'압축해제된 확장 프로그램을 로드합니다 (Load unpacked)'** 버튼을 클릭합니다.
4. 다운로드하거나 클론한 폴더(본 프로젝트 폴더)를 선택하여 등록합니다.

### 3단계: 주소창 우측 고정
1. 크롬 우측 상단의 퍼즐 조각 모양 아이콘(확장 프로그램 메뉴)을 클릭합니다.
2. **YouTube Ad Full Watch** 옆의 핀 아이콘을 눌러 툴바에 고정합니다.
3. 아이콘을 눌러 대시보드를 열고 유튜브를 새로고침하면 대행 작동이 즉시 시작됩니다.
